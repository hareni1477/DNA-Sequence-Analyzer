# DNA SEQUENCE ANLYZER

A menu-driven C++ application for performing basic DNA and protein sequence analysis using fundamental bioinformatics algorithms.

## Features

### Sequence Analysis
- DNA sequence validation
- Nucleotide counting (A, T, G, C)
- GC content calculation
- Reverse complement generation

### Gene Analysis
- Motif search
- Open Reading Frame (ORF) detection

### Protein Analysis
- DNA to protein translation
- Protein length calculation
- Amino acid composition
- Approximate molecular weight calculation

## Concepts Used

- C++
- Strings
- Functions
- Vectors
- Unordered Maps (STL)
- Basic bioinformatics algorithms
